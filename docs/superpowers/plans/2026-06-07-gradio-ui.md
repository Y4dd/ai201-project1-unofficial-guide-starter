> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `app.py` — a Gradio web interface that wires `retrieve()` → `generate()` into a student-facing chat UI, displaying the LLM answer and the source documents it was retrieved from.

**Architecture:** `app.py` at the project root exposes a single `handle_query(question)` function that calls `src.retrieve.retrieve()` for top-5 chunks, then `src.generate.generate()` for a grounded answer. The Gradio `Blocks` layout wraps this function — one textbox for input, one for the answer, one for source document names. Source names come from chunk metadata (reliable) rather than parsed from the LLM response (fragile). An empty-input guard short-circuits before any API call.

**Tech Stack:** Python 3.11+, `gradio>=6.9.0`, existing `src.retrieve` + `src.generate`

- [ ] **Step 3: Create `app.py`**

```python
import gradio as gr

from src.generate import generate
from src.retrieve import retrieve


def handle_query(question: str) -> tuple[str, str]:
    if not question.strip():
        return "Please enter a question.", ""
    chunks = retrieve(question)
    answer = generate(question, chunks)
    sources = "\n".join(f"• {c['source_file']}" for c in chunks)
    return answer, sources


with gr.Blocks(title="WMU Housing Guide") as demo:
    gr.Markdown(
        "# WMU Housing Unofficial Guide\n"
        "Ask any question about on-campus or off-campus housing at Western Michigan University."
    )
    inp = gr.Textbox(
        label="Your question",
        placeholder="e.g. Is the KMetro bus free for WMU students?",
    )
    btn = gr.Button("Ask", variant="primary")
    answer_out = gr.Textbox(label="Answer", lines=10)
    sources_out = gr.Textbox(label="Retrieved from", lines=4)
    btn.click(handle_query, inputs=inp, outputs=[answer_out, sources_out])
    inp.submit(handle_query, inputs=inp, outputs=[answer_out, sources_out])


if __name__ == "__main__":
    demo.launch()
```
